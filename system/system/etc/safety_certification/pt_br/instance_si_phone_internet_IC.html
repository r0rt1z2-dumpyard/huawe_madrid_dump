Compatibilidade no Canadá

Informações sobre certificação (SAR)

Este dispositivo foi projetado para satisfazer aos requisitos de exposição a RF estabelecidos pela Industry Canada.

O limite da SAR adotado pelo Canadá é de 1,6 W/kg sobre 1 g de tecido. O mais alto valor da SAR informado à IC para esse tipo de dispositivo é compatível com este limite.

Declaração de capital intelectual

Este aparelho é compatível com CAN ICES-3 (B)/NMB-3(B).

Este dispositivo é compatível com os padrões Industry Canada licence-exempt RSS standard(s). Sua operação está sujeita às seguintes condições: (1) este aparelho não deve causar interferências; e (2) este aparelho deve aceitar qualquer interferência recebida, incluindo interferência que ocasione operações indesejadas do aparelho.
